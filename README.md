# Insurance Fraud Detection Using Machine Learning

## Project Overview
This project aims to identify potentially fraudulent insurance claims using supervised machine learning techniques.

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Workflow
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Classification model training
- Model evaluation
